Confidence Intervals

Working with means:

1. Confidence interval choosing the margin of error
Explore how different choices for the margin of error affects the "success rate" of the confidence interval process!
2. Confidence interval using normal distribution, sigma known.
Appreciate how confidence intervals built from theory naturally adjust for sample size and population characterstics. For this simulator, we assume that we know the population standard deviation.
3. Confidence interval using normal distribution, sigma unknown
Witness the weakness of using the normal distribution theory to create confidence intervals. When sample sizes are small and we don't know the population standard deviation, our success rate suffers!
4. Confidence interval using the t-distribution
Appreciate how confidence intervals built from theory naturally adjust for sample size and population characterstics. In this case, we do not assume we know the population standard deviation. Unlike with the normal distribution, our success rate stays the same even when sample sizes get small!
